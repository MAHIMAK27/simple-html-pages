Simple Calculator Web App
📌 Overview

This project is a Simple Calculator built using HTML, CSS, and JavaScript. The calculator supports basic arithmetic operations like addition, subtraction, multiplication, and division, along with features such as clearing the screen, adding decimals, and evaluating expressions.

The goal of this project is to provide a simple, responsive, and user-friendly calculator that works directly in the web browser.

🎯 Features
Basic Operations: Addition, subtraction, multiplication, division.
Decimal Support: Add decimal points to numbers.
Clear Button: Resets the current input.
Equals Button: Evaluates and displays the result of the expression.
Responsive Layout: Designed for both desktop and mobile devices.
Error Handling: Displays "Error" when invalid input is encountered.
🖥️ Tech Stack
Frontend:
HTML: For the structure of the calculator.
CSS: For styling and making the calculator responsive.
JavaScript: For handling the functionality and logic behind calculations.
